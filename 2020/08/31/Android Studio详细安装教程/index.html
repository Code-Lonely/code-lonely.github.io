Android Studio详细安装教程

一、下载安装包

  • 点击[这里]下载安装包。

二、准备工作

  • 正式安装之前，最好先准备两个文件夹：一个用来存放安装程序，一个用来存放SDK。效果如下如图，所示。
  • 注意：文件夹路径不要带空格！

三、Android Studio安装

  • 双击打开安装包，正式安装。




  • 耐心等待安装完成，大约2min…
  • 取消打开Android Studio的对勾，切记！

    四、初始化Android Studio配置

  • 打开如下图所示路径下的idea.properties文件
  • 打开之后在如下图所示相应位置添加以下内容
    idea.config.path=D:/Android/.AndroidStudio/config
idea.system.path=D:/Android/.AndroidStudio/system

    五、打开软件Android Studio安装SDK

  • 在开始菜单中找到Android Studio并打开

  • 出现以下界面，选择cancel

  • 选择自定义SDK路径
  • 选择主题
  • 接着选择SDK路径为之前创建的文件夹！

  • 点击Finish
  • 然后开始疯狂跑安装
  • 耐心等待即可
  • 安装成功后，出现以下界面，不要急着创建工程！
  • 先进入设置界面配置一下Gradle的路径
  • 设置gradle路径为：D:/Android/.gradle

    六、换源

  • 软件默认使用官方谷歌源，但由于国内有qiang，所以这里选择国内阿里源
  • 在如下图所示路径中打开相应文件
  • 在对应位置中加入以下代码
    maven { url'https://maven.aliyun.com/repository/public/' }
maven { url'https://maven.aliyun.com/repository/google/' }
maven { url'https://maven.aliyun.com/repository/jcenter/' }
maven { url'https://maven.aliyun.com/repository/central/' }
  • 七、新建工程

  • 项目信息填写完毕之后，点击Finish

  • 之后在界面左侧，找到如下图所示文件

  • 去[这里]下载对应的gradle离线包

  • 将其放到如下图所示对应文件夹

  • 注意：操作完毕后，务必完全关闭软件，再重新开启！！！

  • 等待工程构建完毕

    八、新建模拟器

  • 点击界面右上角的这个图标

  • 选择模拟器型号（手机型号），这里随便选就可以了，选完点击Next

  • 出现以下界面，点击Download

  • 耐心等待下载

  • 下载完成之后，点击Finish
    #九、运行项目

  • 点击页面右上角的启动按钮

  • 耐心等待项目启动….

  • 启动成功界面如下

  • 至此，软件安装成功！
